# starship prompt configuration

Updated 2024-07-10.

## Colors

Allowed colors:

- black
- blue
- cyan
- green
- purple
- red
- white
- yellow

Note that "magenta" maps to "purple" here.

Colors are documented in the advanced configuration section.
